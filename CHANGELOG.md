# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-03-29

### Added

- Initial release with 3 templates:
  - `basic` - Simple CLI with commander
  - `mcp-server` - Full MCP server (stdio mode)
  - `multi-agent` - Orchestrated multi-agent system
- Interactive prompts using inquirer
- Template engine with Handlebars
- MCP client wrapper
- TypeScript by default with strict mode
- Vitest testing setup
- README auto-generation
- Git initialization
- npm publish ready configuration

### Features

- One-command project generation
- Custom template selection
- MCP integration flags
- Skip prompts mode (`--yes`)
- Auto-generated project structure
